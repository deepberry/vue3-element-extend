# Vue3-Element-Extend

一个对 element plus 扩展的组件库，解决部分组件~~太屎~~或延展性太差的问题。<br/>
组件需通过 es6 module 形式引入，本身项目环境需按照[vue-boilerplate](https://github.com/deepberry/vue-boilerplate)进行配置。

## Get Started

### 1.Install

安装前请参考[DeepBerry 私有包安装](https://github.com/deepberry/FE-docs/blob/master/packages/DeepBerry%E7%A7%81%E6%9C%89%E5%8C%85%E5%AE%89%E8%A3%85.md)进行对应设置。

```shell
npm install @deepberry/vue3-element-extend
```

### 2.Import

#### A.全局引入全部组件

```javascript
import DeepBerryUI from "@deepberry/vue3-element-extend";
app.use(DeepBerryUI);
```

#### B.引入单个组件

```javascript
import { UploadImage } from "@deepberry/vue3-element-extend";
app.component("UploadImage", UploadImage);
```
